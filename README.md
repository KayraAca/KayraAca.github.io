# Personal Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, JavaScript, and Bootstrap. Features a dark theme with beautiful animations and a focus on backend development and cryptography.

## üéØ Customization

### Personal Information
Edit the following sections in `index.html`:
- Update name in navbar and hero section
- Modify typewriter phrases in `script.js`
- Change Python code content in the code block
- Update project details and links
- Add your social media links

### Styling
Customize colors in `styles.css`:
```css
:root {
    --violet: #8b5cf6;        /* Primary color */
    --teal: #06b6d4;          /* Secondary color */
    --dark: #0f172a;          /* Background */
    /* ... other variables */
}
```

### Content
- **About section**: Edit personal description and skills
- **Projects**: Update project cards with your actual work
- **Contact**: Add real contact form handling or email links
